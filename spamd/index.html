OpenBSD spamd

spamd(8) представляет из себя демон, блокирующий спам очень эффективным способом.
Он работает с различными базами данных, для настройки которых используется файл /etc/mail/spamd.conf, и поддержку которых обеспечивает spamd-setup(8).

  • U of A Traplist
    Университет Альберты (Канада) использует spamd на своих главных MX (почтовые серверы), и проделывает огромную работу по созданию так называемых серых списков с целью отлова хостов, занимающихся рассылкой. Адреса машин удаляются из списка через 24 часа.

    Адреса OpenBSD зеркал: http://www.openbsd.org/spamd/traplist.gz

  • NiX spam
    NiX spam - список подозрительных адресов, отформатированных для spamd.

    Адреса OpenBSD зеркал: http://www.openbsd.org/spamd/nixspam.gz
