Документация

На Web сайте проекта OpenBSD доступны man страницы для последней версии OpenSMTPD.

  • smtpd - «Simple Mail Transfer Protocol»-демон
  • smtpd.conf - описание формата конфигурационного файла для «Simple Mail Transfer Protocol»-демона
  • smtpctl - консольная программа для управления «Simple Mail Transfer Protocol»-демоном
  • newaliases - настройка базы данных для файлов email-псевдонимов
  • makemap - настройка базы данных доменов при помощи ключей
  • aliases - программа для настройки и установки email-псевдонимов файлов (aliases files format).
  • forward - программа для настройки и установки email-перенаправления (forwarding files format).
Протокол SMTP, реализованный в OpenSMPD, описан в RFC 5321.
