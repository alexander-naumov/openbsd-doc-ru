Портируемая версия OpenNTPD

Основная OpenNTPD представляет из себя небольшую, безопасную и легкую в поддержке версию, разрабатываемую для проекта OpenBSD. Группа разработчиков, занимающаяся портированием OpenNTPD на другие операционные системы, берет эту основную версию и добавляет код, обеспечивающий работу OpenNTPD на других операционных системах.

Портируемая версия OpenNTPD работает на следующих Unix-подобных операционных системах:

  • FreeBSD
  • NetBSD
  • Linux
  • Solaris
  • Mac OS X
  • HP-UX
  • QNX4
  • IRIX
  • AIX

Портируемая OpenNTPD следует разработке официальной версии, но выпуски не синхронизируются. Портируемые релизы помечаются литерой 'p' (например 3.9p1). Официальный исходный код для OpenBSD никогда не будет использовать суффикс 'p'.
